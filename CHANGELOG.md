# Changelog

All notable changes to the AI Behavioral Safety Studies portfolio will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-25

### Added
- Initial portfolio release with comprehensive documentation
- **Case Study 01**: Identity-Based Calibration evaluation
- **Case Study 02**: Legitimacy Framing without Authority evaluation
- **Methodology**: Complete evaluation framework with ethical guidelines
- **Observations**: Synthesized behavioral patterns across studies
- **Tools**: Transcript analyzer Python script for pattern detection
- **Examples**: Sample transcript demonstrating expected patterns
- **Documentation**: README, CONTRIBUTING, LICENSE, and validation scripts
- **Validation**: Portfolio structure and content validation tool
- **Demo**: Demonstration script showcasing portfolio capabilities

### Key Features
- Systematic evaluation of LLM behavioral patterns under social framing
- Quantitative analysis of disclaimer reduction and jargon increase
- Ethical testing framework aligned with AI safety standards
- Reproducible methodology with clear documentation
- Open-source tools for transcript analysis
- Professional portfolio structure suitable for research and presentation

### Technical Details
- All documentation in Markdown format
- Python analysis tool using standard library only
- JSON-based transcript format for interoperability
- Comprehensive validation of portfolio structure
- MIT licensed for open use and contribution

### Known Limitations
- Initial release focused on two primary case studies
- Analysis tool uses regex-based pattern matching (may need customization)
- Sample data is illustrative rather than from actual production systems
- Cross-model validation limited in initial release
